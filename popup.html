how to get your credentials

notion integration token

  1. go to notion integrations
  2. click "new integration"
  3. name your integration (eg, "canvas sync")
  4. select the workspace to install it to
  5. if you want the logos to match, a "512.png" is avalible in the github icons folder for this project. you can upload whatever you'd like tho. it doesn't impact functionality :p
  6. click "save" and copy the "internal integration secret" then paste it in the provided box

notion page ID

  1. open your notion page in a browser
  2. look at the URL: notion.so/workspace/page-id-here
  3. copy the page ID part (typically 32 characters), paste it in respective box, and save
  4. share your page with your integration by clicking "share" and adding your integration

canvas API token

  1. log into your canvas account
  2. go to account → settings
  3. scroll to "approved integrations"
  4. click "New Access Token"
  5. give it a purpose (eg, "canvas sync") and click "Generate Token"
  6. copy the token, paste it and save (you won't be able to see it again!)

canvas base URL

this is simply the main URL for your Canvas instance, for example:

  • https://yourschool.instructure.com
  • https://canvas.university.edu
